Module: business_logic/location

In this module the location calculation is done. For the calculation the type of motion is used and the database is queried for railway, street and building.

(inner) check_RAILWAY(locations, positions, callback)

Function which calculates the location if street and building can be excluded due to the type of motion.
Parameters:
Name Type Description
locations object object with the empty weights for all locations (railway, street, building)
positions Array the array is expected to have the best three positions in it, positions should be the result of choosePositions from positionsHelper
callback function function which will be called with the result of the location calculation, param1 of callback is the error which is null if no error occurred, param2 of callback is an object with id, name, description, weight and allWeights
